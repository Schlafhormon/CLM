# CLM Documentation

This directory contains operational documentation for the standalone CLM tool.

## Start Here

  migration workflow.
- [postcopy_workflow.md](postcopy_workflow.md): manual and CLI-oriented
  post-copy migration workflow.
- [shared_storage_nfs.md](shared_storage_nfs.md): expected shared storage
  layout for CRIU images, run artifacts, and logs.
- [workload_scenarios.md](workload_scenarios.md): Flask workload endpoints and
  load profile behavior.
- [measurement_hygiene.md](measurement_hygiene.md): practical rules for
  repeatable measurements.
- [metrics_reference.md](metrics_reference.md): generated metrics and their
  semantics.
- [downtime_segments.md](downtime_segments.md): downtime segment extraction and
  timeline breakdowns.

Historical notes, archived tuning logs, paper result plots, and local
measurement datasets intentionally remain in the original research repository.
